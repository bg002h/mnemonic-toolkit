# mnemonic-toolkit

Top-level integration CLI: takes a BIP-39 phrase or watch-only xpub and emits a 3-card engraving bundle (ms1 + mk1 + md1).

Installs as binary `mnemonic`.

Documentation: [`design/SPEC_mnemonic_toolkit_v0_1.md`](../../design/SPEC_mnemonic_toolkit_v0_1.md) (root of this repo).
