# `md` GUI forms {#gui-forms-md}

Structural form renders for all 10 subcommands on the **`md`** tab. See the [GUI Forms reference overview](#gui-forms-reference) for how to read a render and what `<masked>`, `(required)`, and `[disabled]` mean. Each form's prose, per-flag reference, and worked example live in its own subcommand chapter, reached from the `> **GUI form:**` cross-link there.
