Verify backup strings re-encode to a given template

Usage: md verify [OPTIONS] --template <TEMPLATE> <STRINGS>...

Arguments:
  <STRINGS>...  

Options:
      --template <TEMPLATE>   
      --key <@i=XPUB>         
      --fingerprint <@i=HEX>  
      --network <NETWORK>     Network for xpub validation [default: mainnet] [possible values: mainnet, testnet, signet, regtest]
  -h, --help                  Print help
