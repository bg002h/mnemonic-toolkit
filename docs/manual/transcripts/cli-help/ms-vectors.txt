Print the SHA-pinned v0.1 test-vector corpus as JSON

Usage: ms vectors [OPTIONS]

Options:
      --pretty
          Indent the JSON output for human readability

  -h, --help
          Print help (see a summary with '-h')

EXAMPLES:
  ms vectors                # compact JSON
  ms vectors --pretty       # indented JSON
  ms vectors | jq '.[0]'    # filter via jq
