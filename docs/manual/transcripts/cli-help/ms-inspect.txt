Inspect an ms1 string's structural fields and decoder verdict

Usage: ms inspect [OPTIONS] [MS1]

Arguments:
  [MS1]
          ms1 string to inspect. Use `-` or omit to read from stdin

Options:
      --json
          Emit JSON instead of text verdict + fields

  -h, --help
          Print help (see a summary with '-h')

EXAMPLES:
  ms inspect <ms1>          # verdict + fields
  ms inspect <ms1> --json   # structured output for tooling
  printf "ms10e ntrsq…" | ms inspect -   # back-typed chunked form
