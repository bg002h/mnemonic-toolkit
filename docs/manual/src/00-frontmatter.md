# About this manual

The m-format star is a family of four sibling Bitcoin self-custody
backup formats that engrave together as a coherent steel-engravable
bundle. This manual is the end-user companion to the four formats and
to the `mnemonic-toolkit` integration tool.

> **CC0 1.0 Universal (Public Domain).** This manual is freely
> redistributable. See `LICENSE` in the source repository.

This is **manual v0.1**, tracking `mnemonic-toolkit` `main` (initial
sync targets toolkit `v0.8.0`). The manual is a *living document*: each
toolkit tag triggers a fresh PDF build attached to the corresponding
GitHub release.
